# PWA修复和网站审计方案

**目标:** 修复PWA播放失败、增加首页分类、添加下载入口、修复CORS问题

---

## Task 1: 调查并修复PWA播放失败

**问题:** 用户在手机PWA上播放影片失败

**可能原因:**

1. CORS问题导致视频无法加载
2. Service Worker缓存策略问题
3. HLS.js配置在PWA模式下不兼容
4. HTTPS/WSS连接问题

**修复方案:**

- [ ] 检查PWA模式下的网络请求
- [ ] 验证HLS.js在PWA环境下的配置
- [ ] 测试视频源在PWA模式下的访问
