# 5572 移动端UX全面改进计划

> **目标**: 基于顶级APP设计模式，全面改进移动端体验

## 核心改进

### 1. 首页 - Netflix风格

**当前问题**: 横向滑动找片体验差，内容被遮挡

**改进方案**:
```
┌─────────────────────────┐
│     全屏Hero大图         │  ← 自动轮播，大标题+播放按钮
│     [立即播放]           │
├─────────────────────────┤
│  热门电影  更多 >        │  ← 分类标题
│  ┌───┐ ┌───┐ ┌───┐     │
│  │   │ │   │ │   │ →   │  ← 横向滚动（只在分类内）
│  └───┘ └───┘ └───┘     │
├─────────────────────────┤
│  热播剧集  更多 >        │
│  ┌───┐ ┌───┐ ┌───┐     │
│  │   │ │   │ │   │ →   │
│  └───┘ └───┘ └───┘     │
├─────────────────────────┤
│  🏠  🔍  ⭐  📥  👤   │  ← 底部Tab
└─────────────────────────┘
```

### 2. 播放页 - 沉浸式体验

**当前问题**: 控制栏遮挡内容，操作不直观

**改进方案**:
```
┌─────────────────────────┐
│                         │
│      视频播放区域        │  ← 全屏沉浸
│      (手势控制)          │
│                         │
├─────────────────────────┤
│  片名 - 第X集           │
│  [选集] [投屏] [设置]   │  ← 简洁操作栏
├─────────────────────────┤
│  猜你喜欢               │
│  ┌───┐ ┌───┐ ┌───┐    │  ← 推荐内容
│  └───┘ └───┘ └───┘    │
└─────────────────────────┘
```

### 3. 搜索页 - 极简高效

**当前问题**: 搜索框太小，结果展示不清晰

**改进方案**:
```
┌─────────────────────────┐
│  🔍 搜索影视内容...     │  ← 大搜索框
├─────────────────────────┤
│  热门搜索               │
│  [鱿鱼游戏] [三体] ...  │  ← 热门标签
├─────────────────────────┤
│  搜索结果               │
│  ┌───┐ ┌───┐           │
│  │   │ │   │           │  ← 网格结果
│  └───┘ └───┘           │
└─────────────────────────┘
```

### 4. 个人中心 - 简洁列表

**改进方案**:
```
┌─────────────────────────┐
│  👤 用户名              │  ← 头像+用户名
│  会员状态               │
├─────────────────────────┤
│  ⭐ 我的收藏      >    │
│  📥 离线缓存      >    │
│  📜 播放历史      >    │
│  ⚙️ 设置         >    │  ← 功能列表
│  ❓ 帮助反馈      >    │
└─────────────────────────┘
```

## 技术实现

### 文件结构
```
src/ui/mobile/
├── layouts/
│   ├── MobileLayout.tsx      # 基础布局
│   └── MobileNav.tsx         # 底部导航
├── components/
│   ├── MobileHeroBanner.tsx  # 全屏Hero
│   ├── MobileVideoCard.tsx   # 视频卡片
│   ├── MobileContentSection.tsx  # 内容区块
│   ├── MobileSearchBar.tsx   # 搜索框
│   └── MobilePlayerControls.tsx  # 播放控制
├── pages/
│   ├── MobileHomePage.tsx    # 首页
│   ├── MobilePlayerPage.tsx  # 播放页
│   ├── MobileSearchPage.tsx  # 搜索页
│   └── MobileProfilePage.tsx # 个人中心
└── hooks/
    └── useMobileGesture.ts   # 手势Hook
```

## 实施顺序

1. **Phase 1**: MobileHeroBanner + MobileContentSection（核心组件）
2. **Phase 2**: MobileHomePage（首页）
3. **Phase 3**: MobilePlayerPage（播放页）
4. **Phase 4**: MobileSearchPage（搜索页）
5. **Phase 5**: MobileProfilePage（个人中心）
6. **Phase 6**: 集成到路由系统
7. **Phase 7**: 测试验证
